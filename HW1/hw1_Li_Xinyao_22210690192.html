2、第二问

令lambda=1.8，station 1中服务时间服从均值为1的指数分布， station 2中服务时间服从均值为0.5的指数分布。利用排队论的相关理论可以计算，顾客在整个排队系统中的逗留时间的期望为10.26左右。按照题中要求，先对仿真系统进行3000单位时间的warm-up，后采集1000个样本计算其平均逗留时间，得到结果为11.3306，与理论值差距较大。可能的原因是采集样本数量较少。将样本数量扩大至100000，计算100000个样本的平均逗留时间得到结果为10.1138，该结果与理论值10.26较为接近。画出样本逗留时间的频率分布直方图如下：
该问具体实验代码如下：
n = 100000;
lambda = 1.8;
mu1 = 1;
mu2 = 2;
rng(11);
tarr1 = cumsum(exprnd(1/lambda,1,n));
tsrv1 = exprnd(1/mu1,1,n);
tsta1 = zeros(1,n);%接受服务的时刻
tlea1 = zeros(1,n);%离开station 1的时刻
twat1 = zeros(1,n);%顾客在station 1的总时间
last1 = [0 0];%服务台结束当前服务的时刻
for i = 1:n
%找出最快结束服务的服务台时刻
[minemp, k] = min(last1);
%比较服务台最快能够提供下一次服务的时刻与下一位顾客到来的时刻以得到开启下一次服务的时刻
tsta1(i) = max(tarr1(i),minemp);
tlea1(i) = tsta1(i) + tsrv1(i);
last1(k) = tlea1(i);
twat1(i) = tlea1(i) - tarr1(i);
end
%按离开节点1的时间先后顺序对顾客进行排序
[tarr2, order] = sort(tlea1);
tsrv2 = exprnd(1/mu2,1,n);
tsta2 = zeros(1,n);%顾客在station 2接受服务的时刻
tlea2 = zeros(1,n);%顾客离开station 2的时刻
twat2 = zeros(1,n);%顾客在station 2的时间
last2 = 0;%服务台结束服务时刻
for i = 1:n
minemp = last2;
tsta2(i) = max(tarr2(i),minemp);
tlea2(i) = tsta2(i) + tsrv2(i);
last2 = tlea2(i);
twat2(i) = tlea2(i) - tarr2(i);
end
twat2(order) = twat2;
[rev,ooderr] = sort(order);
twat = twat1+twat2;
Index = min(find(tarr1>3000));
sample = twat(Index:Index+999);
h1 = histogram(sample,50,'Normalization','probability');
title("histogram of the sojourn time for (b)");
%3000秒warm-up后取1000个样本求平均值
sample_average_time_for_b = sum(sample)/1000
sample_average_time_for_b = 11.3306
%100000个样本求得的平均值
total_average_time_for_b = sum(twat)/n
total_average_time_for_b = 10.1138

3、第三问

若station 2中服务者的服务时间服从均值为0.5、标准差为0.15的正态分布，可以通过理论计算得到顾客在整个排队系统中的逗留时间的期望为8.22左右（因为此处需要计算形如：、等积分。在计算此类积分时，无法求得其精确解，在计算数值解时会产生系统误差、舍入误差等误差，因此此处理论计算得到的结果8.22也并非精确值。）按照题中要求，先对仿真系统进行3000单位时间的warm-up，后采集1000个样本计算其平均逗留时间，得到结果为6.7539，与理论值差距较大。可能的原因是采集样本数量较少。将样本数量扩大至100000，计算100000个样本的平均逗留时间得到结果为8.1259，该结果与理论值10.26较为接近。画出样本逗留时间的频率分布直方图如下：
该问具体实验代码如下：
rng(11);
n = 100000;
lambda = 1.8;
mu1 = 1;
mu2 = 2;
tarr1 = cumsum(exprnd(1/lambda,1,n));
tsrv1 = exprnd(1/mu1,1,n);
tsta1 = zeros(1,n);%接受服务的时刻
tlea1 = zeros(1,n);%离开station 1的时刻

for i = 1:n
%找出最快结束服务的服务台时刻
[minemp, k] = min(last1);
%比较服务台最快能够提供下一次服务的时刻与下一位顾客到来的时刻以得到开启下一次服务的时刻
tsta1(i) = max(tarr1(i),minemp);
tlea1(i) = tsta1(i) + tsrv1(i);
last1(k) = tlea1(i);
twat1(i) = tlea1(i) - tarr1(i);
end
%按离开节点1的时间先后顺序对顾客进行排序
[tarr2, order] = sort(tlea1);
tsrv2 = zeros(1,n);
for i = 1:n
r=normrnd(0.5,0.15);
while r<0
r=normrnd(0.5,0.15);
end
tsrv2(i)=r;
end
tsta2 = zeros(1,n);%顾客在station 2接受服务的时刻
tlea2 = zeros(1,n);%顾客离开station 2的时刻
twat2 = zeros(1,n);%顾客在station 2的时间
last2 = 0;%服务台结束服务时刻
for i = 1:n
minemp = last2;
tsta2(i) = max(tarr2(i),minemp);
tlea2(i) = tsta2(i) + tsrv2(i);
last2 = tlea2(i);
twat2(i) = tlea2(i) - tarr2(i);
end
[rev,ooderr] = sort(order);
%计算第i位到达station 1的顾客在该系统中的总时间

Index = min(find(tarr1>3000));
sample = twat(Index:Index+999);
h2 = histogram(sample,50,'Normalization','probability');
title("histogram of the sojourn time for (c)");
sample_average_time_for_c = sum(sample)/1000
sample_average_time_for_c = 6.7539
total_average_time_for_c = sum(twat)/n
total_average_time_for_c = 8.1259
